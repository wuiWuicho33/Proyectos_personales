Tipos de datos

1.- Valores

La sintaxis de JavaScript define dos tipos de valores:

Valores fijos Valores variables
Se denominan literales y existen dos tipos:
  • Números
  • Cadenas
 Se denominan variables

2.- Literales

Las variables pueden contener muchos tipos de datos, pero el más común son los literales y existen de dos tipos, números y cadenas.

Números

Los números se pueden escribir con o sin decimales:

100
10.5

Cadenas

Las cadenas son texto, escrito entre comillas simples o dobles.

Se escriben sin comillas, pero de colocarse con ellas se interpretarán como una cadena de texto.

"Mario"
'Mario'
"20"

3.- Expresiones

Una expresión es una combinación de valores, variables y operadores que se calcula en un valor.
El cálculo se llama evaluación.

Las expresiones pueden contener valores literales y variables.
